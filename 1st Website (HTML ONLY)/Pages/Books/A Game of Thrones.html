Book Description

In "A Game of Thrones," it's a wild ride from a frosty, tough land to a lush, feast-filled kingdom. You've got lords, ladies, warriors, and wizards, plus sneaky assassins and tough-as-nails bastards, all mixing it up when some super dark signs start showing up. It's all about power plays, backstabs, and epic battles!

Author: George R. R. Martin

Number of Pages: 694
